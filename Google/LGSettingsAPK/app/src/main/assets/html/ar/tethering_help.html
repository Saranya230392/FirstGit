USB اتصال

لمشاركة الانترنت من جهازك مع جهاز الكمبيوتر. USB يمكنك وصل جهازك الأندرويد بجهاز الكمبيوتر عن طريق إتصال
  • تعمل نقطة Wi-Fi المحمولة مع إصدارات Windows Vista, Windows 7 , Linux

  • أذا كان يحتوي جهازك على بطاقة SD أو ذاكرة داخلية لا يمكنك إزالة تركيبها عندما يكون إتصال USB فعالا

  • لمزيد من المعلومات حول اتصال USB مع أنظمة التشغيل الأخرى يرجى زيارة :
    http://www.android.com/tether/usb
