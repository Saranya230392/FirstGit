La Conexión Wi-Fi Portátil te permite usar tu celular con tecnología Android como una conexión móvil para hasta cinco aparatos usando la conexión de datos de tu teléfono. Si estás usando este servicio, deberías asegurar la conexión para restringir el acceso no autorizado. Este servicio gasta la reserva de batería y aumenta tu uso de datos.

Este servicio requiere una suscripción al plan de módem correspondiente. Por favor asegúrate de suscribirte al plan de datos apropiado antes de usarlo. Llama al 1-888-944-9400 si necesitas asistencia.

El uso del servicio constituye la aceptación de los términos del módem para Smartphone.
